# Professional Portfolio Website

A modern, responsive portfolio website built with HTML, Tailwind CSS, and JavaScript. Features smooth animations, interactive elements, and a professional design.

## 🌟 Features

### Design & UI
- **Modern Design**: Clean, professional layout with gradient backgrounds and glass effects
- **Responsive**: Fully responsive design that works on all devices
- **Smooth Animations**: CSS animations and transitions throughout the site
- **Interactive Elements**: Hover effects, click interactions, and dynamic content
- **Professional Typography**: Using Inter font family for clean readability

### Pages
1. **Home Page** (`index.html`)
   - Hero section with animated text
   - Featured projects showcase
   - Skills preview
   - Call-to-action sections

2. **About Page** (`about.html`)
   - Personal story and background
   - Professional timeline
   - Values and principles
   - Interactive profile section

3. **Projects Page** (`projects.html`)
   - Project filtering by category
   - Interactive project cards
   - Project modals with details
   - Search functionality
   - Statistics section

4. **Skills Page** (`skills.html`)
   - Detailed skill categories
   - Animated progress bars
   - Skill filtering and search
   - Certifications section
   - Learning journey timeline

5. **Contact Page** (`contact.html`)
   - Contact form with validation
   - Contact information cards
   - FAQ section
   - Social media links
   - Interactive map placeholder

### Technical Features
- **Form Validation**: Real-time validation with error messages
- **Smooth Scrolling**: Navigation and anchor link smooth scrolling
- **Mobile Menu**: Responsive mobile navigation
- **Loading Animations**: Intersection Observer for scroll-triggered animations
- **Back to Top**: Smooth scroll to top functionality
- **Local Storage**: Form auto-save functionality
- **Accessibility**: Keyboard navigation and ARIA labels

## 🚀 Getting Started

### Prerequisites
- A modern web browser
- No build tools required - pure HTML, CSS, and JavaScript

### Installation
1. Clone or download the repository
2. Open `index.html` in your web browser
3. The website is ready to use!

### File Structure
```
portfolio/
├── index.html          # Home page
├── about.html          # About page
├── projects.html       # Projects page
├── skills.html         # Skills page
├── contact.html        # Contact page
├── js/
│   ├── main.js         # Main JavaScript functionality
│   ├── projects.js     # Projects page specific JS
│   ├── skills.js       # Skills page specific JS
│   └── contact.js      # Contact page specific JS
└── README.md           # This file
```

## 🎨 Customization

### Personal Information
Update the following in each HTML file:
- Name: Replace "John Doe" with your name
- Email: Update contact email addresses
- Phone: Update phone number
- Location: Update your location
- Social Media: Update social media links

### Colors and Styling
The website uses a purple-blue gradient theme. To customize:
1. Update the gradient colors in the CSS variables
2. Modify the `gradient-bg` class in the style sections
3. Update button colors and hover states

### Content
- **Projects**: Add your own projects to the projects page
- **Skills**: Update skills and proficiency levels
- **About**: Personalize your story and experience
- **Contact**: Update contact information and social links

### Images
- Replace placeholder icons with actual project images
- Add your profile photo to the about page
- Update favicon and meta images

## 📝 Usage

### Navigation

## 🛠️ Development

2. Style with Tailwind CSS classes
3. Add JavaScript functionality

- Modify CSS keyframes in the style sections
